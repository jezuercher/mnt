eba_default_options

PURPOSE

[options,constraints] = eba_default_options(network)

SYNOPSIS

function [options,constraints] = eba_default_options(network)

DESCRIPTION 

 [options,constraints] = eba_default_options(network)

 set default values for structures 'options' and 'constraints' 
 used to define an eba task

 constraints.z:           linear weights in the objective function  
 constraints.v_fix:       vector predetermined fluxes
 constraints.v_min:       vector of lower bounds
 constraints.v_max:       vector of upper bounds
 constraints.v_sign:      predetermined flux signs
 constraints.ext_sign:    sign vector for external metabolite production
 constraints.mu_fix:      given mu values        
 constraints.mu_min:      lower bounds for mu values  
 constraints.mu_max:      upper bounds for mu values
 constraints.dmu_fix:     given delta mu values        
 constraints.dmu_min:     lower bounds for delta mu values  
 constraints.dmu_max:     upper bounds for delta mu values  

 options.seed             random seed
 options.compute_mu       'mu', 'delta_mu'
 options.verbose
 options.optimality_criterion 'z','minimum_norm'
 options.n_trials          repeat eba from differemt starting points, take best result
